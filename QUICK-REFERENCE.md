# Quick Reference: Homepage Sections

## 🎯 What You Can Add

### 1. Hero Sliders
- **Location:** Top of page
- **Admin Tab:** Hero Sections
- **Features:** Auto-play carousel, 4 types
- **Recommended:** 3-5 images per slider

### 2. Category Icons  
- **Location:** Below hero
- **Admin Tab:** Category Icons
- **Features:** 7 icon categories with hover
- **Icons:** monitor, printer, cloud, fingerprint, card, smartphone, idcard

### 3. Feature Cards
- **Location:** Below categories
- **Admin Tab:** Feature Cards
- **Features:** Large promo cards, custom colors
- **Recommended:** 2-4 cards

### 4. Brand Logos
- **Location:** Below feature cards
- **Admin Tab:** Brands
- **Features:** Logo grid, grayscale hover effect
- **Recommended:** 6-12 brands

### 5. Partnerships/Certificates
- **Location:** Below brands
- **Admin Tab:** Partnerships
- **Features:** Certificate images, descriptions
- **Layout:** Alternating left/right

---

## 🚀 Quick Setup (5 Minutes)

1. **Run SQL:**
   ```bash
   # In Supabase SQL Editor:
   - create-category-feature-tables.sql
   - create-brands-partnerships-tables.sql
   ```

2. **Go to Admin:**
   ```
   https://yourdomain.com/admin
   ```

3. **Configure Sections:**
   - Hero Sections → Add slider with 3-5 images
   - Category Icons → Edit default 7 categories
   - Feature Cards → Add 2 promo cards
   - Brands → Add 6 partner logos
   - Partnerships → Add 1 certificate

4. **Done!** 🎉

## ⚡ Pro Tips

1. **Images:** Use high-res (1920px+ for hero)
2. **Brands:** Use official logos with transparent backgrounds
3. **Certificates:** Scan at 300 DPI minimum
4. **Mobile:** Test on real devices
5. **Performance:** Optimize all images before upload
